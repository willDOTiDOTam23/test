# Lexus LFA White Symphony

A single-page showcase for the Lexus LFA "White Symphony" hero section. The project is a
static HTML + CSS layout focused on typography, dramatic imagery, and a grid-based feature
breakdown.

## Project goals

- Present the Lexus LFA story with a bold hero section and supporting feature panels.
- Keep the build lightweight with zero dependencies.
- Make the layout responsive for modern desktop and mobile widths.

## Tech stack

- HTML5
- CSS3 (grid + modern typography via Google Fonts)

## Getting started

1. Open `index.html` in your browser.
2. Edit `index.html` or `styles.css` to customize copy or styles.
